🎯 GESTURE BASED VOLUME CONTROL

---

1. INTRODUCTION

Gesture Based Volume Control is a computer vision-based project that allows users to control system volume using hand gestures through a webcam. This system provides a touchless and intuitive way of interacting with a computer.

---

2. OBJECTIVE

The main objective of this project is to design a real-time system that detects hand gestures and converts them into volume control operations.

---

3. SOFTWARE REQUIREMENTS

- OpenCV
- NumPy
- MediaPipe
- PyAutoGUI
- PyCharm (Development Environment)

---

4. HARDWARE REQUIREMENTS

- Processor: Intel i5 or above (64-bit OS)
- RAM: Minimum 4GB
- Webcam (Required)
- 64GB Storage

---

5. METHODOLOGY

- Capture video using webcam
- Detect hand landmarks using MediaPipe
- Calculate distance between fingers
- Map finger distance to volume level
- Adjust system volume accordingly

---

6. OUTPUT

The system successfully detects hand gestures and controls the system volume in real time.



---

7. HOW TO RUN

1. Install required libraries:

pip install -r requirements.txt

2. Run the project:

python main.py
